# What to do in Paris

## Project Purpose
This project aims to help users discover cultural events in Paris and make it easy to reach them from the Ecla dorm in Noisy-le-Grand.

## Tech Stack
- **FastAPI**
- **React (Vite)**
- **MUI**
- **Diskcache**
- **Docker**

## Highlighted Features
- **Repository Pattern Architecture**
- **Distance Calculation**: Utilizes the Haversine formula. Currently, it uses the author's home location but could be adapted to use the user's home/current location from the database in a real project.
- **PRIM API Integration**: Provides real-time travel duration.
- **API Caching Layer**: Enhances performance by leveraging caching mechanisms.
